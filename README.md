# CineAnalytica

An End-to-End Movie Intelligence Platform built using Python, SQL, Machine Learning, NLP, and Streamlit.

## Project Modules

- 📊 EDA Explorer
- 🎬 Box Office Revenue Predictor
- 💬 Sentiment Analyzer
- 🎯 Hybrid Recommendation Engine

Tech Stack:
Python | SQLite | Scikit-learn | XGBoost | NLP | Streamlit
